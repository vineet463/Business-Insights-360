# Business Insights 360

### Project Overview

A comprehensive Power BI Dashboard built using SQL and Power BI to provide a overall view of financial, sales, marketing, supply chain, and executive performance.

The dashboard enables stakeholders to track KPIs, monitor profitability, analyze customer and product performance, and improve decision-making through interactive analytics.

### Business Problem

AtliQ Hardware has been growing rapidly and expanding into global markets. To keep up with competitors who already rely on advanced analytics, the company decided to leverage Power BI for making data-driven decisions.

This dashboard answers key stakeholder questions across:

- Finance
- Sales
- Marketing
- Supply Chain
- Executive

### Data Source

The dashboard gathers data from two primary sources:

-Excel/CSV Files
-MySQL Database

The data is stored in SQL databases and connected directly to Power BI for reporting.
The project uses two databases provided in the Bootcamp: gdb041 and gdb056.
Together, they cover customers, products, markets, sales, forecasts, and associated costs.

### Tools 

- Power BI Desktop
Used to build interactive dashboards, data visualization, modeling, and reporting.

- Power Query
Used for data cleaning, transformation, shaping datasets, and creating pipelines.

- DAX
Used to create measures, KPIs and perform complex calculations.

- Data Modelling
Designed relationships, star schema concepts, optimized performance, ensured accurate aggregations.

- MySQL Server
Used as primary data source for storing transactional and master data.

- Excel
Used for quick validations, data checks, and exploratory data analysis.
